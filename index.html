HOW TO PLAY

Defend your space guitar against incoming meteors by playing the correct guitar string that matches the string name on each meteor!

  • Play the string shown on each meteor to fire a laser and destroy it
  • Each destroyed meteor earns points
  • If a meteor hits your ship, you lose a shield
  • The game gets faster and more challenging as you progress
  • Make sure your guitar is connected and your microphone is allowed

This game helps you learn and memorize guitar string names while having fun!
